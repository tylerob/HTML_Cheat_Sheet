Cheat Sheet for Common HTML Tags

Made by Tyler O'Brien
Jan. 2024

CSS Tags

Tag Name Description
<link rel="stylesheet" href="..."> CSS Link Link to External or Internal Stylesheet
<div style="..."> Inline Style Format text inline on the index HTML
<div class="example"> Class Identification Create a class for a group of text to be edited in CSS
<div id="example"> ID Identification Create an ID for a group of text to be edited in CSS; supercedes Class

Media Tags

Tag Name Description
<a href="..."> Anchor Creates a hyperlink to pages or files with the nested text as a link
<video width=".." height=".." controls> Video Creates embedded video with set size and controls. Src required.
<video src="..."> Video Embeds selected video from source

Table Tags

Tag Name Description
<table> Table Wrapper element for all HTML tables
<thead> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table data container
<tfoot> Table Foot The set of rows defining the table footer

Table Attributes

Tag Name Description
<colspan> Column Span Defines how many columns a td element should span
<rowspan> Row Span Defines how many rows a td element should span
